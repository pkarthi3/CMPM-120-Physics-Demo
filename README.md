All visual assets were drawn by me in IbisPaint X. The frog sprite is based on a similar image by Irwan Johnny on Vecteezy (https://www.vecteezy.com/vector-art/49681176-cute-frog-cartoon-isolated-on-white-background).
The game can be played at https://pkarthi3.itch.io/cmpm-120-physics-game.

Requirements:
- Continuous and discrete player inputs: Moving the character in the levels is continuous; going between levels on the results screen is discrete
- The player's goal can only be achieved indirectly through the physics engine: While the player does have control over the player character, it isn't possible to just move the player character straight to the goal as this is a platformer style game
- 3+ gameplay scenes: The game has 3 distinct levels
- Other scenes separate and contextualize the gameplay scenes: There is a results screen after each level
